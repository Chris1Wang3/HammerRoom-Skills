# 用户输入模板

## 极简输入

```text
原话/会议纪要：
紧急程度：
输出格式：
```

## 推荐输入

```text
1. 老板/业务原话（verbatim）：
2. 汇报/对齐对象（老板 / 业务 / 研发负责人）：
3. 紧急程度（今天回话 / 本周方案 / 下 sprint）：
4. 行业 & 产品形态：
5. 已知约束（人力 / 预算 / 合规 / 技术栈）：
6. 成功标准（对方口头说过什么算交差）：
7. 明确不做什么：
8. 交付档位（L1 / L2 / L3，默认 L2）：
9. 输出格式（Markdown / HTML，默认 Markdown）：
10. 是否接着做评审预演（是 / 否，默认否）：
```

## 示例 1：权益激励

```text
老板说：最近新客转化不太好，看看能不能搞点权益激励。
我现在只有这句话，帮我先拆成能跟老板确认的问题和方案骨架。
```

## 示例 2：且慢/盈米新手引导

```text
业务说：能不能给新用户做一个进阶引导，别一上来就流失。
我还没拿到完整需求，先帮我拆成对齐问题、核心场景和一期方案骨架。
```

## 示例 3：会议纪要

```text
会议纪要里只有一句：投顾服务要提升触达后的承接效率。
帮我先拆一下，这到底要确认什么、可能做哪些方向、怎么跟业务对齐。
```
