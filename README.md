# Jiaye-Skills

> Product manager skill collection for AI agents — competitive research and PRD review simulation, both with professional HTML outputs.

## Skills

| Skill | Directory | What It Does |
|---|---|---|
| **Competitive Product Research** | `competitive-product-research/` | Dual-track competitive analysis (experience benchmarking + strategic diagnostics), source-traceable HTML report, action-focused output |
| **PRD Review Simulator** | `requirement-review-simulator/` | 5-role cross-functional stress-test, deterministic survival scoring, meeting-ready HTML review report |

## Quick Start

```text
# Competitive research
Our app's post conversion rate is only 3%. Compare with Xiaohongshu and Instagram.

# PRD review
Review my membership-system-2.0 PRD. Realistic mode.
```

## Install

```bash
# Install individual skills from ClawHub
npx clawhub@latest install competitive-product-research
npx clawhub@latest install requirement-review-simulator
```
